# Mipango App

## Overview

Mipango is an objective-driven productivity application designed to help users turn their goals into clear, actionable plans. Instead of focusing only on tasks, Mipango guides users on what to do next based on their objectives, time, and energy.

---

## Problem Statement

Most to-do list applications focus on task management rather than goal achievement. Users often:

* Create long task lists without clear priorities
* Struggle to decide what to work on next
* Lose motivation due to lack of direction
* Fail to connect daily tasks to long-term goals

Mipango solves this by shifting focus from tasks to objectives and providing intelligent guidance.

---

## Solution

Mipango helps users:

* Define clear objectives
* Break objectives into manageable tasks
* Prioritize tasks based on importance and urgency
* Receive suggestions on what to work on next
* Adapt plans based on progress and constraints

---

## Target Users

* Students and apprentices (e.g., Zone01 learners)
* Early-career professionals
* Freelancers and self-managed workers
* Anyone working towards structured goals

---

## Core Features (MVP)

### 1. Objective Creation

Users can:

* Define a goal
* Set a deadline
* Specify priority level

### 2. Task Management

* Create tasks linked to objectives
* Mark tasks as complete
* View tasks by objective

### 3. Smart Task Suggestion

* Suggest next task based on:

  * Priority
    n  - Deadline
  * Progress

### 4. Progress Tracking

* Show completion percentage per objective
* Track completed vs pending tasks

### 5. Simple Planning View

* Daily or weekly task recommendations

---

## Future Features (V2+)

* AI-powered objective breakdown
* Energy-based task recommendations
* Adaptive scheduling
* Reflection and feedback system
* Trade-off analysis between objectives

---

## User Flow

1. User creates an objective
2. User adds tasks related to the objective
3. App suggests what to work on next
4. User completes tasks
5. App updates progress and adjusts suggestions

---

## Value Proposition

Mipango is not just a to-do list.

It is a system that helps users:

* Focus on what truly matters
* Make better decisions about their time
* Stay aligned with their goals

---

## Tech Considerations

* Database: (e.g., PostgreSQL or SQLite)
